# PYTHON MODULES FOR  CONTENT PRODUCTION CHAIN

## INTRODUCTION:
 In this directory we get every module that follow structure of:
 ```bash
 script_runner.sh
 script.py
 requirements.txt
 readme.md
 ```

 **script_runner.sh** have role of verify that all dependencies and libraries required for script are avaible, then it run the script

 **requirements.txt** have the standard versions for every dependencie and library in each module

 **script.py** have the current source code for the module, it have a basic testing for get sure that there's all the avaible resources after run the script inside the main pipeline
