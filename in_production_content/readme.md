#INTRODUCTION
 This directory have the role of divide the several places required to process audio, video and editions of the whole production chain **probaly the most important directory** because here we'll manage most data and files for our **content production chain**


##QUICK VIEW OF EVERY SUBDIRECTORY:

### *audio* directory:
 This directory manage the role of get the base audio at **first stage of production** when we separate **raw video** and we get **raw audio** in this place audio we'll be processed for **enhancement stage**

### *data_sets* directory:
 This directory contains the **core data for our whole production chain** exactly **time stamps data sets** and all data sets in *.pkl* and *.json* files for **complementary videos**, **complementary songs**, **shorts timestamps for extraction**

### *downloaded_music* directory:
 This directory contains **complementary music downloaded for videos**

### *enhanced audios* directory:
 This directory contains our **raw audio that was processed for clean voice AI module** for better quality

### *complementary videos* directory:
 This directory contains videos that was downloaded for **Pexels videos downloader** 

### *transcriptions* directory:
 This directory contain **main text resource** for get all **required data for our workflow**, exactly our video transcription

### *enhanced_audios* directory:
 This directory contains **raw audio post cleanvoiceAI python module**

### *enhanced_videos* directory:
 This directory contains **raw video post FFMEG VIDEO enhancement** 

### *mixed_videos* directory:
 This directory contains result of mixed **video and audio** once both of this are **enhanced**

### *videos_with_music* directory:
 This directory contains result of **enhanced video with music** 

### *video_with_subtitles* directory:
 This directory contains result of **enhanced video with generated subititles**
